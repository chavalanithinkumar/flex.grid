🧠 Quick Learning Guide

Flex Direction

Controls whether items go left→right (row) or top→bottom (column).

Justify Content

Controls how items are positioned along the main axis.

Space Between

Pushes items apart with equal space between them.

Grid Columns

Controls how many columns the grid has.

Gap

Adds spacing between grid items.
